--------------WEM-------------

- WEM - Expirience Manegment

- Ferramenta que fornece recurso para gerenciamento de conteudo
  sem a necessidade de conhecer programação.

----------Vantagens do WEM--------- 

- Gestao de Conteudos Dinamicos

- Caheamento de componentes da pagina 

- Escalabilidade da aplicação

------MGMT-------

CMS - Content Manegment Server - Servidor de Gerenciamento de conteudo

-------Dentro do CMS-----------

VCN Admin - Servidor de Aplicação do WEM/Vingnett porta: 27055

VCM Server - Runtime Service - Deploy do WEM - porta: 27001 

Deploy Agent - Da - Serviço de Persistencia no Banco de Dados e Conexao com o Banco de Dados - porta: 27155

Node Manager - Serviço que permite a comunicação entre o VCN Admin e outros nodes para startar VMs em outros computadores - porta: 27155

------LIVE/PRODUÇÃO--------

CDS - Content Delivery Server - Servidor de entrega de conteúdo

----------Dentro do CDS----------
Libs
Binarios
Cfg agent
Deploy Agent - Da - Serviço de Persistencia no Banco de Dados e Conexao com o Banco de Dados

-------Servidor de Preview-------

- Responsavel pela apresentação

-------Dentro Servidor de Preview-------

DSM - Dynamic Site Module -  Integração do WEM/Vingnett com o Preview

Servidor de Aplicação Ex: Jboss,TomCat EE,Websphere, Web Logic - Roda o vgn-ext-templating.war "Motor do WEM"
